## [1.0.0] - 2026-02-20

### Added

- Single-page portfolio website for Dann — Software Architect
- Hero section with enterprise modernization and AI-integrated systems positioning
- Dashboard section with impact metrics
- Staff Mastery section (skills and expertise)
- Projects section: Elis Eternity, Fintech AI, Blockchain (Nativas)
- AI Lab section: Repo Architect, n8n automation flows
- Leadership section
- Tech Stack (Arsenal) section
- Soft Radar section (emerging technologies)
- Contact CLI — terminal-style contact interface with easter egg
- Social Hub — LinkedIn, GitHub, HackerRank, WhatsApp, Discord
- Internationalization (i18n): English and Spanish via `locales/en` and `locales/es`
- Dark/light theme with random initial theme and manual toggle
- Navbar with language switcher and theme toggle
- Footer with CV downloads (AI/Dev, EN/ES variants)

### Technical

- Vanilla JS (ES modules), no build step
- Tailwind CSS (CDN), Inter + Space Grotesk fonts, Font Awesome icons
- Anime.js for animations
- BackgroundEngine, PerformanceManager, ScrollController, SectionRenderer
- Dev tools (localhost): Ctrl+Q quality cycle, Ctrl+S stats, Ctrl+I device info, Ctrl+D debug scroll
